# Prompt Engineering Documentation

## Overview

StackSave uses structured prompt generation for executive AI summaries.

The prompt architecture was designed to:
- remain concise
- generate actionable insights
- focus on optimization opportunities
- summarize cost-saving recommendations

---

# Prompt Structure

The AI receives:
- tool stack data
- recommendations
- savings calculations

and generates:
- executive-level summaries

---

# Example Prompt

```txt
You are an AI spend optimization expert.

Analyze this AI tooling stack.

Generate a concise executive summary explaining:
- overspending risks
- overlapping tooling
- optimization opportunities
- estimated savings impact
